# Crypto Arbitrage IDE

Desktop-приложение на Python с тёмной IDE-темой для мониторинга и сканирования арбитражных спредов.

## Как теперь работает сканер
- Программа собирает глобальный universe монет по всем подключенным spot-биржам.
- В цикл попадает до **10000 уникальных кодов монет**.
- Сканирование идёт **пакетами по 50 монет** без повторов, пока не закончится полный проход.
- После завершения полного прохода начинается новый цикл.
- Для каждой монеты программа ищет:
  - самую дешёвую биржу (`BUY`)
  - самую дорогую биржу (`SELL`)
  - возможность перевода именно **с BUY на SELL**
- В основную таблицу и окно топа добавлены компактные колонки:
  - `BUY`
  - `SELL`
  - `TX` (`YES`, `?`, `NO`)
- Клик по `PAIR` открывает сразу две страницы: биржу покупки и биржу продажи.

## Возможности
- Динамический выбор бирж (включая BingX)
- Сопоставление монет только по **точному коду монеты**
- Проверка перевода именно между самой дешёвой и самой дорогой биржей
- Режим `Strict transfer only`:
  - исключает `UNVERIFIED`
  - показывает только связки с подтвержденной совместимой сетью перевода
- Окно сохранённого топа:
  - хранит до 10 лучших монет
  - позволяет исключить ошибочную монету крестиком `x`
  - исключённая монета не вернётся в топ до конца текущего сеанса
  - окно масштабируется и можно уменьшать сильнее, чем раньше
- Подсветка min/max цены (зелёный/красный)
- Кликабельные цены (переход на конкретную биржу)
- Фильтры по % разницы, TOP N, сортировка
- Режим Turbo (20x10)
- Память настроек: `user_settings.json`
  - `Сохранить` / `Загрузить`
  - автосохранение при закрытии

## Установка
```bash
python -m venv .venv
.venv\\Scripts\\activate
pip install -r requirements.txt
```

## Запуск
```bash
python app.py
```

## Примечания
- На некоторых биржах отдельные пары могут отсутствовать: в таблице будет `N/A`.
- Для автообновления используйте интервал от 10 секунд и выше.
