# Crypto Arbitrage IDE

Desktop-приложение на Python с тёмной IDE-темой для мониторинга и сканирования арбитражных спредов.

## Как теперь работает сканер
- Программа загружает до **3000 spot-монет с Bybit**.
- Сканирование идёт **пакетами по 50 монет** без повторов, пока не закончится весь цикл.
- После завершения полного прохода начинается следующий цикл.
- Для каждой монеты программа ищет:
  - самую дешёвую биржу
  - самую дорогую биржу
  - возможность перевода **именно с биржи покупки на биржу продажи**
- В общее окно сохраненного топа добавляются **5 лучших монет из текущего отображаемого batch**.
- В памяти держатся только **10 лучших** сохраненных монет.

## Возможности
- Динамический выбор бирж (включая BingX)
- Сопоставление монет только по **точному коду монеты**
- Проверка маршрута перевода:
  - спред считается между самой дешёвой и самой дорогой биржей
  - проверяется вывод с дешёвой биржи и депозит на дорогую
  - в таблице показывается колонка `ROUTE`
  - если сети не раскрыты обеими биржами, используется `UNVERIFIED`
- Режим `Strict transfer only`:
  - исключает `UNVERIFIED`
  - показывает только связки с подтвержденной совместимой сетью перевода
- Подсветка min/max цены (зелёный/красный)
- Кликабельные цены (переход на пару в браузере)
- Фильтры по % разницы, TOP N, сортировка
- Режим Turbo (20x10)
- Память настроек: `user_settings.json`
  - `Сохранить` / `Загрузить`
  - автосохранение при закрытии
- Окно сохраненного топа масштабируется и может быть уменьшено сильнее, чем раньше

## Установка
```bash
python -m venv .venv
.venv\\Scripts\\activate
pip install -r requirements.txt
```

## Запуск
```bash
python app.py
```

## Примечания
- На некоторых биржах отдельные пары могут отсутствовать: в таблице будет `N/A`.
- Для автообновления используйте интервал от 10 секунд и выше.
