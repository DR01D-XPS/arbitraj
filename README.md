# Crypto Arbitrage IDE

Desktop-приложение на Python с тёмной IDE-темой для мониторинга и сканирования арбитражных спредов.

## Как теперь работает сканер
- Программа загружает **полный список spot-монет с Bybit**.
- Далее идёт по этому списку **пакетами по 50 монет** (каждое обновление — новый batch).
- Для каждого batch считает цены на выбранных биржах и спред.
- Если монета доступна **на всех выбранных биржах**, она попадает в накопительный пул.
- Когда в пуле набирается 50 таких монет:
  - берётся **топ-5 по % разницы**,
  - эти 5 монет добавляются в **одно общее окно сохраненного топа**,
  - окно синхронизируется с настройками основного (выбранные биржи, котировка, turbo),
  - в памяти держатся только **10 лучших** сохраненных монет,
  - окно продолжает обновляться дальше,
  - основной сканер идёт к следующим 50 монетам.

## Возможности
- Динамический выбор бирж (включая BingX)
- Проверка совпадения активов:
  - приоритет по `contract address`, если он доступен у биржи
  - защита от ложных совпадений по одинаковому тикеру
  - повторный поиск монеты на других биржах по найденному контракту, если тикер отличается
- Проверка маршрута перевода:
  - спред считается только если найден совместимый маршрут вывода/депозита
  - в таблице показывается колонка `ROUTE`
  - если сети у бирж не раскрыты публично, может использоваться `MANUAL CHECK`
- Подсветка min/max цены (зелёный/красный)
- Кликабельные цены (переход на пару в браузере)
- Фильтры по % разницы, TOP N, сортировка
- Режим Turbo (20x10)
- Память настроек: `user_settings.json`
  - `Сохранить` / `Загрузить`
  - автосохранение при закрытии

## Установка
```bash
python -m venv .venv
.venv\\Scripts\\activate
pip install -r requirements.txt
```

## Запуск
```bash
python app.py
```

## Примечания
- На некоторых биржах отдельные пары могут отсутствовать: в таблице будет `N/A`.
- Для автообновления используйте интервал от 10 секунд и выше.
